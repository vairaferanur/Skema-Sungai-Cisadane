# Skema-Sungai-Cisadane
Skema jaringan Sungai Cisadane yang disusun berdasarkan klasifikasi orde sungai.

## Deskripsi
Repository ini hasil kegiatan magang yang dilaksanakan di bawah koordinasi Dinas Komunikasi dan Informatika Provinsi Jawa Barat, dengan penugasan di Dinas Sumber Daya Air Provinsi Jawa Barat. Skema Sungai Cisadane menampilkan struktur jaringan sungai berdasarkan klasifikasi orde sungai. Skema disusun secara skematik menggunakan AutoCAD sebagai media visualisasi untuk mendukung analisis jaringan sungai.

## Kegiatan
- Pengolahan dan penyederhanaan data jaringan sungai
- Klasifikasi orde sungai
- Penyusunan skema jaringan sungai

## Tools
AutoCAD

## Output
Skema jaringan Sungai Cisadane dalam bentuk peta
